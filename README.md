# Twitter Sample App

| Gem | Version |
| --- | --- |
| Ruby | 2.7.5 |
| Rails | 6.1.4.1 |

This is the sample application for
[*Ruby on Rails Tutorial:
Learn Web Development with Rails*](https://www.railstutorial.org/)
(6th Edition)
by [Michael Hartl](http://www.michaelhartl.com/).


## Process

```
$ bundle install --without production
```
```
$ rails db:migrate
```
```
$ rails s
```
```
$ rails c --sandbox (dev)
```
